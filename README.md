# AddUserFullStack

A full-stack web app with:
- 🔧 Spring Boot (Backend)
- 💻 ReactJS (Frontend)
- 🛢️ MySQL (Database)

---

## 🚀 How to Run

### 🔙 Backend (Spring Boot)
1. Open `backend` folder in IDE
2. Update `application.properties` with your DB config
3. Run the Spring Boot App

### 🎨 Frontend (ReactJS)
1. `cd frontend`
2. `npm install`
3. `npm start`
